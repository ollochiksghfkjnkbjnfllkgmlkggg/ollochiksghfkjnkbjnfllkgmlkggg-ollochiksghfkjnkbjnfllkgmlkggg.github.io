Мягкая игрушка «Утята» — это набор из двух мягких игрушек в виде маленьких утят. Они выполнены из мягкого и приятного на ощупь материала, который безопасен для детей. Они помогут развить у детей чувство ответственности за живое существо, а также научат их заботиться о других.
