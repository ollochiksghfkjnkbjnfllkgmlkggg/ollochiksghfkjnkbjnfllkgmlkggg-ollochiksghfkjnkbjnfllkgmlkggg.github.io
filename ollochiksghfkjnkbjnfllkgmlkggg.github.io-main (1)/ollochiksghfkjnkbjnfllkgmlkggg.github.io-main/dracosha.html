ДРАКОША
1299р.
Мягкая игрушка «Дракоша» — это забавная игрушка в виде маленького дракона. Она выполнена из мягкого и приятного на ощупь материала, который безопасен для детей. Внутри игрушки находится гипоаллергенный наполнитель, который не вызывает аллергических реакций. Мягкая игрушка «Дракоша» станет отличным подарком для ребёнка или любителя фантастических существ. Она поможет развить у детей чувство ответственности за живое существо, а также научит их заботиться о других.
